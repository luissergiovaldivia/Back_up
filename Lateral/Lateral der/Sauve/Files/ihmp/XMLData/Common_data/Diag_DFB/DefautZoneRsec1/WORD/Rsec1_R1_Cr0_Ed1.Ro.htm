Releul de securitate nu se armeaza dupa anulare defect si rearmare

 

 

Releul de securitate este HS

Verificati releul ,dupa anulare defect si rearmare

  • Daca este alimentat si nu anclanseaza, inlocuiti contactorul.
  • Daca nu, analizeaza cauza urmatoare

Releul care insumeaza informatiile de securitate este HS

 

Verificati releul, dupa anulare defect si  rearmare

  • Daca este alimentat si nu anclanseaza, inlocuiti releul
  • Daca anclanseaza si nu furnizeaza tensiune, inlocuiti releul
  • Daca anclanseaza si  furnizeaza tensiune ,analizati cauza urmatoare

 

Iesirea AP este HS

 

Verifica iesirea de pe modulul IBS, dupa anulare defect si rearmare

  • Daca este alimentata si nu furnizeaza tensiune dupa anulare defect si rearmare, inlocuiti modulul
  • Daca nu, analizati cauza urmatoare
