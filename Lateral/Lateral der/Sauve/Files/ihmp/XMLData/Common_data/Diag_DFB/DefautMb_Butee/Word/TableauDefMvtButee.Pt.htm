Defeito movimento stopper : Os sensores de abertura (DOB) e fechamento (DFB) são a 0 ou a 1 simultaneamente.

 

 

1-     Se os dois sensores são a 0

 

Causas

Solução

O stopper esta preso mecanicamente

Se possivel, eliminar a causa mecanica do bloqueio.

Se não : chamar a manutenção.

Um dos sensores esta fora de serviço

 

Se o stopper está fechado, controlar o detector DFB e trocá-lo se necessário.

Se o Stopper está aberto, controlar o detector DOB, e trocá-lo se necessário.

Se o problema persiste: chamar a manutenção.

 

 

1-     Se os dois sensores são a 1.

 

Causas

Solução

Um dos sensores está fora de serviço.

Se o stopper está fechado, controlar o detector DOB, e o trocar se necessário.

Se o stopper está aberto, controlar o sensor DFB, e o trocar se necessário.

Se o problema persistir: chamar a manutenção.
