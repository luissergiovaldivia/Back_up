During the fault, PLC set safety outputs, while contactor power presence check is de-activated

 

 

Problem

Solution

The safety contactor is not activated (coil isn’t power supplied)

Check safety contactor coil supply

·          If not power supplied, check wiring

·      Otherwise, jump to next problem.

Safety Contactor is out of use

 

Check safety contactor

·          If contactor is not closed while activated, replace contactor

·          Otherwise, jump to next problem.

PLC input, safety contactor status check, is out of use

Expected Status = 0

 

Check digital input on I/O device

·          If not power supplied, replace the module

·          Otherwise, jump to next problem.

A voltage (24v or other ) is present in safety contactor status check

Check wiring

 

ATTENTION, IF TWO HAND CONTROL(S):

AFTERWARDS CORRECTION OF FAULT, TO ACTIVATE THE SAFETY OUTPUT,

PUSH IN THE SAME TIME

·         ON ALL THE TWO HAND CONTROL PUSHBUTTONS

·         ON THE CONTROL ON PUSHBUTTONS OF THE FACILITY OR CYCLE START PUSHBUTTON OF THE ZONE
