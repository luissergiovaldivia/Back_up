Référence déjà présente dans la base de données

 

Causes

Remèdes

Dans la base de données il y a un numéro de référence identique à un autre.

Vérifier que :

  - pour chaque emplacement dans la base de données (BDD), la valeur du mot de référence n'est pas égale à la valeur de l’entrée référence (REF).
