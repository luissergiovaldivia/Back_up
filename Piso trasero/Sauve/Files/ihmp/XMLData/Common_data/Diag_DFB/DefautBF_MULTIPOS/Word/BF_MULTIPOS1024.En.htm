Causes

Solutions

Defect in the position measurement system:   

 

 * #Diag.CodeDef = 1024

 

  

- The measurement system is defective

 

 

For a range finder:

- check that the target is correctly positioned.

- check the 24V power supply or the profinet connection of the range finder.

- check whether the range finder and the target are aligned.

- change the range finder

 

For a bar code encoder:

- check the 24V power supply or the profinet connection of the range finder.

   - check that the bar code strip is legible and is not torn

- check whether the beam is aligned to the bar code strip

- change the encoder
