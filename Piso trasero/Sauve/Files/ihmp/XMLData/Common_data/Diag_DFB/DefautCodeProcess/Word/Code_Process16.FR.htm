Remèdes

Défaut format des données dans table d’entrée :

 

Le paramètre table d’entrée (TAB_ENT) de la BF n’est pas au bon format ou n’est pas situé dans la bonne zone mémoire de l’automate.

Pour l’entrée table d’entrée (TAB_ENT), vérifier que :

- L’entrée de la BF est renseignée.

- Le type de donnée est conforme.

ARRAY : Tableau d'un même type de données simple ou complexes (voir liste ci-dessous).

STRUCT : Structure de plusieurs types de données simples ou complexes (voir liste ci-dessous).

 

Type de données

Description

WORD

Mots (16 bits)

INT

Entiers (16 bits)

DWORD

Mots (32 bits)

DINT

Entiers (32 bits)

 

- L’entrée est située dans une zone mémoire conforme.

Zone

Description

DB

Bloc de données

DI

Bloc de données d’instance
