Zone anti-intrusion check devices fault while support (skid, pallet, hanger,…) is entering or leaving safety area

 

 

Problem

Solution

Anti-intrusion check device, xxcaint1, wasn’t de-activated when support was moving into protected area.

 

Check anti-intrusion device is working properly

·          If not, tune or replace devices

·          Otherwise, jump to next problem.

Problem due to wiring, a voltage (short-circuit) is activating digital input

Check wiring

·          If wrong connection, correct it

·          Otherwise, jump to next problem.

Input Device is out of use

Check digital input where is connected xxcaint1 anti-intrusion device

·          Replace the module
