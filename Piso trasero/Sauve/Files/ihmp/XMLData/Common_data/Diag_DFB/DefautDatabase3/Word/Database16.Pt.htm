Referência inexistente dentro da base de dados (leitura, modificação ou exclusão)

 

Causas

Soluções

A pesquisa dentro de cada base de dados não retornou nenhum número de referência equivalente ao número de referência documentado na entrada (REF).

 

Verificar se:

   - a entrada OffRef corresponde à localizalção da palavra de referência dentro de uma vaga (TALO)

   - a totalidade da base de dados (BDD) foi consultada (NbMaxEt = NbMaxEt da função de escritura)
