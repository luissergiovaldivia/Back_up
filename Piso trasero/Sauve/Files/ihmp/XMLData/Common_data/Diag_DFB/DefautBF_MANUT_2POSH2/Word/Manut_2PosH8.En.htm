Defect in forward overtravel:

 

* #Diag.CodeDef = 8

 

* #Diag.Ihmdfsurca at 1

 

- The turn table or cross transfer is in Forward overtravel.

 

 

- Console

1- Select the movement, select the "Manual " Mode

 

2- Using the "reverse" pushbutton, reposition the cross transfer or the turn table

 

3- Delete the defect and select the "Auto " mode

 

- Check the operation of the low speed forward movement detector
