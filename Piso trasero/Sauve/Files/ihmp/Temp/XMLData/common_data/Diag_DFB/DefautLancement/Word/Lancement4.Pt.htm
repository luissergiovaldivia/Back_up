Entrada TAB_MAP : Zona de memorização incorreta: IhmInfo = 34

Entrada TAB_MAP : Vaga muito pequena: IhmInfo = 35

 

O parâmetro de entrada ‘’Tabela MAP’’ da FB não possui uma boa dimensão.

A dimensão do parâmetro ANY ‘’Tabela de troca’’ depende do parâmetro ‘’Tipo de tabela’’

O parâmetro ‘’tabela de troca’’ não deve ser menor que os valores abaixo.

 

TABTyp = 1 : 117 palavras

TABTyp = 2 : 88 palavras

TABTyp = 3 : 120 palavras

TABTyp = 7 : 120 palavras

TABTyp = 10 : 240 palavras

Entrada TAB_MAPEx : Vaga muito pequena: IhmInfo = 38

 

O parâmetro de entrada ‘’Tabela MAP estendida" da FB não possui uma boa dimensão.

Esta entrada é utilizada apenas para tabelas de troca 10(BCE10).

O parâmetro ‘’Tabela MAP estendida’’ não deve ser menor que o valor abaixo.

 

TABTyp = 10 : 50 palavras

 

Entradas TAB_MAP e TAB_MAPEx incorretas: IhmInfo = 39

 

Os parâmetros de entrada ‘’Tabela MAP’’ e ‘’Tabela MAP estendida’’ não foram informados.

 

1) Verificar o parâmetro ‘’TABTyp’’. Este contrôle não é feito para tabelas de troca n°10 (BCE10)

2) Preencher pelo menos um das entradas: ‘’TAB_MAP’’ e/ou ‘’TAB_MAPEx’’
