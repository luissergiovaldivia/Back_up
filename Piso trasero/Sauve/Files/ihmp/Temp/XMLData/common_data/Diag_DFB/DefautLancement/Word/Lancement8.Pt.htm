Causes

  Defeito de lançamento MAP

 

Causas

Soluções

Defeito Tabela MAP: IhmInfo = 49

A tabela MAP não contém valores positivos.

Este controle é realizado para todas as tabelas de trocas (TABTyp).

 

Particularidade se ‘’TABTyp’’ = 10 :

Se nenhuma variável não for documentada na entrada TAB_MAP, o controle não será realizado e nenhum defeito irá aparecer.

 

A tabela MAP deve conter ao menos um valor não nulo.

 

Defeito tabela MAP estendida: IhmInfo = 50

A tabela MAP estendida não contém valores positivos.

Este controle não será realizado para as tabelas de troca n°10 (TABTyp = 10)

 

A tabela MAP estendida deve conter ao menos um valor não nulo.
