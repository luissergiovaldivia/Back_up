Référence inexistante dans la base de données (lecture, modification ou effacement)

 

Causes

Remèdes

La recherche dans chaque étiquette de la base de données n'a pas permis de trouver un numéro de référence équivalente au numéro de référence documenté en entrée de boîte (REF).

Vérifier que :

   - l’entrée OffRef correspond à l’emplacement du mot de référence dans une étiquette (TALO)

   - l’intégralité de la base de données (BDD) a été consultée (NbMaxEt = NbMaxEt de la fonction écriture)
