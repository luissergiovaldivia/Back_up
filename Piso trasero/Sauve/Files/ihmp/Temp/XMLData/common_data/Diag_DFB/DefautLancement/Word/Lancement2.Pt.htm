Defeito de diálogo

 

Causas

Soluções

Diálogo incorreto: IhmInfo = 17

O valor dos dados ‘’valcal’’ e ‘’valapi’’ do bloco de troca não correspondem a um dialogo conhecido.

Lista de diálogos possíveis: VALCAL / VALAPI

VALCAL = 1  / VALAPI = 0 : O calculador propõem uma peça

VALCAL = 2  / VALAPI = 0 : O calculador não propõem nenhuma peça

VALCAL = 0  / VALAPI = 2 : O CLP responde ao pedido proposto pelo calculador

VALCAL = 0  / VALAPI = 1 : O CLP fez a peça proposta

 

1) Verificar o valor das palavras ‘’VALCAL’’ et ‘’VALAPI’’ da tabela de troca

2) Se ‘’VALCAL’’ não conforme: Chamar o operador SIPTOL para correção (valor não suportado pelo ponto de lançamento)

3) Se ‘’VALAPI’’ não conforme: Erro de programação BF_LANCEMENT (modificação do código do programa)

 

Diálogo não iniciado: IhmInfo = 18

Os dados ‘’valcal’’ e ‘’valapi’’ do bloco de troca estão em zero.

1) Usando uma tabela de animação, preencha os valores a seguir dentro da tabela de troca SIPTOL:

                VALCAL = 0    VALAPI = 1    ETFLUX = 0

2) Chamar o operador SIPTOL para retirar o alarme existente na tela IHM SIPTOL.
