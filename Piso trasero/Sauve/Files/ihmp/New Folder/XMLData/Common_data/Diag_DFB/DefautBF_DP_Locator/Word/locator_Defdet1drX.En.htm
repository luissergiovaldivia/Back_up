Causes

DRUM X Limit switch 1 fault: M_Diag = 2

Problem

Solution

The movement is blocked mechanically.

Select zone MANUAL mode and press RETURN and ADVANCE button to find out origin of blocking.

If blocking cause is not visible: ask for maintenance.

The sensor is out of range

If the LED is switched off while part to detect is in front of sensor, check sensor detection by positioning a metal part close to the sensor.

If LED lights, tune the sensor and check its functioning by moving part in both direction in manual mode.

The sensor is faulty

If the LED doesn’t light by moving closer a metal part, Replace the sensor.

If any change: ask for maintenance.
