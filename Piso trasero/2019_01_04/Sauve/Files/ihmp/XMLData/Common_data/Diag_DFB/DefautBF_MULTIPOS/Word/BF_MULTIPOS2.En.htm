Position Loss Defect without control: 

 

  * #Diag.CodeDef = 2

 

  

- After the End of Forward or Reverse Movement, the PLC controller detected and saved the position which was reached but this position was lost later without any request for a new movement.

 

 

1- Check:

- That the range finder value is non zero and the target is detected

- That the tolerance value in the measurement is not too limited

- That the stoppage anticipation value enables positioning at the centre of the control window

- That stoppage occurs when the low speed is stabilised (LS anticipation)

 

 

2- The position is Controlled once again:

- Cancel the Defect
