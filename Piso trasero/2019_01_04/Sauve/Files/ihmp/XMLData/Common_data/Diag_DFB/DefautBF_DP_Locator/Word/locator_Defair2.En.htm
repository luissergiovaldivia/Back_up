Valve isn’t activated, while pressure switch = 1

 

Problem

Solution

Line valve is power supplied (24 V)

Check line valve power supply

  • If voltage is present on power supply, fix it
  • Otherwise, jump to next problem

Soft air valve is out of use

Check air valve

  • If air is passing while line valve not activated, replace it
  • Otherwise, jump to next problem

Pressure range is too low

Check tuning of pressure switch

  • If not tuned at right level of pressure, tune it
  • If whatever tuning, pressure switch is activated, jump to next problem

A voltage (24V) is present between pressure switch and PLC digital input

Check pressure switch wiring

  • If voltage is present, fix it
  • Otherwise, jump to next problem

Pressure switch is out of use

Check pressure switch

  • If there is no air in red and pressure switch output activated, replace it
  • Otherwise, jump to next problem

Pressure switch PLC digital input is out of use, expected at 0

Check PLC digital input

  • If digital input = 1, replace module
