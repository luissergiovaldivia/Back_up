Défaut Sur-Course Haut levage :

 

* M_Diag [2] x4

 

- L’élévateur n’est pas sur la position Basse. L’automate ne voit plus à 1 le fin de Sur-Course Haut.

.

 

 

 

1- Pupitre Elévateur

- Sélectionner le mode manuel

- Appuyer sur le BP Descente si pupitre avec BP Mo /De levage et Av / Re Rouleaux

- Sinon Sélectionné le mouvement levage

- Appuyer sur le BP Recul.

- Relâcher le BP si le Voyant Contrôle Position est allumé

- Sélectionner le mode Automatique
