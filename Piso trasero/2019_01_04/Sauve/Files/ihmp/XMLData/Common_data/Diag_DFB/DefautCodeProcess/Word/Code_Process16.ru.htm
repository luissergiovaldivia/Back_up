Ошибка формата данных при вводе в таблицу :

 

Вводимые параметры таблицы (TAB_ENT) в BF не в правильном формате или находятся не в той памяти  контроллера.

Для ввода параметров в таблицу (TAB_PARAM), проверьте, что :

- Установлен вход в BF.

- Тип данных соответствует.

ARRAY : Список того же типа данных: простых или сложных. (см. ниже).

STRUCT : Структуры нескольких типов данных: простых или сложных (см. ниже).

 

Тип данных

Описание

WORD

Слов (16 bits)

INT

Целые числа (16 bits)

DWORD

Слова (32 bits)

DINT

Целые числа (32 bits)

 

- L’entrée est située dans une zone mémoire conforme.

Зона

Описание

Образец блока данных
