Remèdes

Défaut longueur table d’entrée :

La table d’entrée (TAB_ENT) a une dimension inférieure à ce qui est paramétré par l’entrée Lgcf.

 

Note : La comparaison se fait entre la longueur de la table d’entrée (exprimée en octets) et la valeur renseignée sur l’entrée Lgcf auquel il faut retirer 2 octets représentant un code process.

- Modifier le paramètre d’entrée Lgcf pour correspondre à la table d’entrée

Lgcf : Correspond aux nombres de critères de comparaison plus 2 (octets) pour le code process associé.

          (exemple : 50 mots dans TAB_PARAM => Lgcf = 102 (100 octets + 2 pour code process)

 

- Modifier la table d’entrée (TAB_ENT) pour correspondre à la longueur d’une configuration (Lgcf) (-2 octets).
