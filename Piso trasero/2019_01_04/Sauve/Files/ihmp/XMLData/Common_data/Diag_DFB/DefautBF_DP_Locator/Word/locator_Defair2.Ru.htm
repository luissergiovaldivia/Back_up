Клапан не активирован, переключатель на = 1

 

Проблема

Решение

Есть питание клапана линии (24 В)

Проверьте питание клапана

  • Если есть напряжение, произведите наладку
  • В противном случае, см следующую проблему

Воздушный клапан с гибкими вставками не работает

Проверьте воздушный клапан

  • Если есть утечка воздуха, когда клапан линии не подключен, замените его  
  • В противном случае, см следующую проблему

Диапазон давления слишком низкий

Проверьте настройку переключателя давления

  • Если переключатель настроен неправильно, настройте его
  • Если переключатель не работает, см следующую проблему

Напряжение (24V) есть между переключателем давления и вводом данных PLC

Проверьте проводку переключателя давления

  • Если есть напряжение, произведите наладку
  • В противном случае см следующую проблему

Переключатель давления не работает

Проверьте переключатель давления

  • Если воздух не отмечен красным и вывод переключателя давления активирован, замените его   
  • В противном случае см следующую проблему

Переключатель PLC ввода цифровых данных не работает, ожидается на уровне 0

Проверьте PLC ввод цифровых данных

  • если ввод цифровых данных = 1, замените модуль
