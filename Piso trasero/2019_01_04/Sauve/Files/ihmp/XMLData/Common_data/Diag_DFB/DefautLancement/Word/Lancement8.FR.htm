Causes

  Défaut lancement mode mise au point

 

Causes

Remèdes

Défaut table de mise au point : IhmInfo = 49

La table de mise au point ne contient pas de valeurs positives.

Ce contrôle est effectué pour toutes les tables d’échanges (TABTyp).

 

Particularité si ‘’TABTyp’’ = 10 :

Si aucune variable n’est documentée sur l’entrée TAB_MAP, le contrôle n’est pas effectué et aucun défaut ne sera remonté.

 

La table de mise au point doit contenir au moins une valeur non nulle.

 

Défaut table de mise au point étendue : IhmInfo = 50

La table de mise au point étendue ne contient pas de valeurs positives.

Ce contrôle n’est effectué que pour les tables d’échanges n°10 (TABTyp = 10)

 

La table de mise au point étendue doit contenir au moins une valeur non nulle.
