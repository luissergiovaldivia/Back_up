Defect in reverse overtravel:

 

* #Diag.CodeDef = 16

 

- The displacement reached the Reversing overtravel detector.

 

 

- Console

1- Select the movement, select the "Manual Drive" Mode

 

2- Using the "Forward" push button ,  disengage the overtravel detector

 

3- Delete the defect and select the "Auto Drive" mode
