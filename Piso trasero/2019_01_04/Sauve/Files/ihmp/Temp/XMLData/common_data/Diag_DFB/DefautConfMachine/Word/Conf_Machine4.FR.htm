Défaut format des données dans table de sortie :

 

Le paramètre table de sortie (TAB_SORT) de la BF n’est pas au bon format ou n’est pas situé dans la bonne zone mémoire de l’automate.

Pour l’entrée table de sortie (TAB_SORT), vérifier que :

- L’entrée de la BF est renseignée.

- Le type de donnée est conforme.

ARRAY : Tableau d'un même type de données simple ou complexes (voir liste ci-dessous).

STRUCT : Structure de plusieurs types de données simples ou complexes (voir liste ci-dessous).
