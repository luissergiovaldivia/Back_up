Remèdes

Défaut configuration inconnue :

L’entrée table de paramétrage (TAB_PARAM) ne contient pas la configuration recherché par la table d’entrée (TAB_ENT).

 

L’information Codesse indique le numéro de sous-ensemble ayant activé l’erreur.

Identifier le sous-ensemble n’ayant pas la configuration recherchée (Codesse) puis :

 

  - Pour ce sous-ensemble ajouter ou modifier une configuration dans la table de paramétrage (TAB_PARAM) pour correspondre à la table d’entrée (TAB_ENT).

 

  - Modifier la table d’entrée (TAB_ENT) pour correspondre à une configuration présente dans chaque sous-ensemble de la table de paramétrage (TAB_PARAM).
