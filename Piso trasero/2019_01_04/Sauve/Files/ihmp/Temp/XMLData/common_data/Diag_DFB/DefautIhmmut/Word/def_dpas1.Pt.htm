Defeito detecção de passagem do suporte (luge, gancho…) durante a transferência na entrada ou saída da zona de segurança

 

 

Causas

Soluções

O sensor de passagem, xxdpas1, não ativou durante a transferência do suporte pela zona protegida.

Verificar o funcionamento dos sensores de passagem

·          Se existir um mau funcionamento, regular ou trocar o sensor.

·          Se não, passar para a causa seguinte.

Problema de cabeamento, a informação do sensor xxdpas1, não chegou até o CLP.

Verificar o cabeamento

·          Se mal conectado, arrumar.

·          Se não, passar para a causa seguinte.

Módulo de entrada HS

Verificar a entrada do CLP sobre a qual é conectado o sensor de passagem

·          Se existir um mau funcionamento, trocar o módulo de entrada.
